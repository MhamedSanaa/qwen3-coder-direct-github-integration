# Super Mario Style Side-Scrolling Game

A simple Super Mario-inspired side-scrolling platformer game built with Python and Pygame.

## Features

- Player character that can move left/right and jump
- Platforms to jump on
- Enemies that move back and forth
- Collectible coins
- Side-scrolling camera that follows the player
- Gravity and collision detection

## Controls

- **Left Arrow**: Move left
- **Right Arrow**: Move right
- **Spacebar**: Jump
- **Escape**: Quit game

## Requirements

- Python 3.x
- Pygame library

## How to Run

1. Make sure you have Python installed
2. Install Pygame if you don't have it already: `pip install pygame`
3. Run the game: `python mario_game.py`

## Game Elements

- Red rectangle: Player character
- Green rectangles: Platforms
- Brown squares: Enemies
- Yellow circles: Collectible coins
- Blue background: Sky

The game features a scrolling level that extends beyond the initial screen, allowing for exploration to the right. Collect as many coins as you can while avoiding enemies!
